# arg-count-action
Simple GitHub Action Demo <br>
<br>
This action prints out the number of arguments passed in

## Inputs
**arguments-to-count** <br>
<br>
Required The arguments to count

## Outputs
**argcount** <br>
<br>
The count of the arguments

## Example Usage
uses:&nbsp;thejug/arg-count-action@v1 <br>
with: <br>
&nbsp;&nbsp;arguments-to-count: <arguments>
